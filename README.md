# Pneumonia Detection using ResNet18

## Problem Statement
Classify Chest X-ray images as:
- NORMAL
- PNEUMONIA

## Dataset
Chest X-Ray Images (Pneumonia)

## Model
- ResNet18 (Transfer Learning)
- PyTorch

## Results
- Validation Accuracy: 100% (16-image validation set)
- Test Accuracy: 82.69%

## Technologies Used
- Python
- PyTorch
- Torchvision
- Google Colab
